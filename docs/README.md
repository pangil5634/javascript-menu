# 기능 명세서

## 1. 코치 이름 입력 받기

- [X] 1-1. 입력 받기
- [X] 1-2. 이름 분리하기
- [X] 1-3. 예외 처리하기
  - [X] 미입력 예외 : 아무것도 입력하지 않은 경우
  - [X] 인원 예외1 : 2명 미만인 경우
  - [X] 인원 예외2 : 5명 초과인 경우
  - [X] 이름 예외1 : 2글자 미만인 경우
  - [X] 이름 예외2 : 4글차 초과인 경우
- [X] 1-4. 코치 객체 생성하기

<br>

## 2. 코치 별 메뉴 입력 받기

- [X] 2-1. 입력 받기
- [X] 2-2. 메뉴 분리하기

- [X] 2-3. 예외 처리하기
  - [X] 길이 예외 : 2개 초과인 경우
  - [ ] 미존재 예외 : 존재하지 않는 메뉴인 경우

- [X] 2-4. 코치 별 못 먹는 메뉴 정보 저장하기

<br>

## 3. 메뉴 추천하기

- [X] 3-1. 카테고리 뽑기

- [ ] 3-2. 메뉴 뽑기
- 총 5번 진행

- [ ] 3-2. 코치 별 추천 메뉴 저장하기
